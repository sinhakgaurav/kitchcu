## Shape language

- Rounded / bubble typography → large border-radii (`999px` pills, 16–24px cards)
- Friendly mascot → warm empty states, never corporate cold
- Dark-first UI matches neon/dark lockups

## Casing

Always present brand as **kitchCU** (kitch lowercase, CU uppercase).
